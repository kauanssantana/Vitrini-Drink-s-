# Vitrini-Drink-s-
Plataforma web completa de e-commerce e gestão para a Vitrini Drink's, desenvolvida em PHP e MySQL. Possui área do cliente para compras e um painel administrativo robusto com controle de acesso (Admin/Estoquista) para gerenciar pedidos, usuários, produtos e estoque. Interface moderna, segura e com foco em usabilidade.
