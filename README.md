🚀 Como rodar o projeto localmente
Instale o XAMPP (ou servidor equivalente com Apache e MySQL).

Clone ou baixe este repositório para dentro da pasta htdocs do XAMPP.

Abra o XAMPP Control Panel e inicie os módulos Apache e MySQL.

Acesse o localhost/phpmyadmin no navegador.

Crie um banco de dados chamado cadastroteste.

Clique em Importar e selecione o arquivo cadastroteste.sql que está na raiz do projeto.

Acesse o sistema pelo navegador em localhost/nome-da-pasta-do-projeto.

No caso a tela cliente é index.html e a tela de login para gerenciamento de adm/estoquista é login.php

Para acessar o painel administrativo logar com o e-mail "admin@gmail.com" e com a senha "123456"
